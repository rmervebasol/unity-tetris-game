**# Unity Tetris Game**

**This project is a classic Tetris game developed using Unity and C#.**

**## Features**

**- Score system to track player performance**

**- Game over logic based on gameplay conditions**

**- Visual effects to enhance player experience**

**- Sound effects integrated for in-game actions**

**- All game assets and UI elements designed by me using Adobe Photoshop**

**## Technologies Used**

**- Unity**

**- C#**

**- Adobe Photoshop**
